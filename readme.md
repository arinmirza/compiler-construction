# Compiler Construction

This repository contains my notes and implementations for the *Compiler Construction* course in 2026. It is currently a work in progress.

## Folders
- The compiler implemented in Typescript is under the `compiler` folder.
- The experimental implementation in egglog[^1] is under the `eggs` folder.
- Interesting stuff worth discussing and the progress of the project is documented in `journal.md`

## Usage
```bash
deno compiler/tests/basic-test.ts
```
```bash
egglog eggs/regex.egg
```
